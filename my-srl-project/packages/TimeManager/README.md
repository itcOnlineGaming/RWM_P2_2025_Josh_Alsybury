# Time Manager Component

A Svelte component for time management with Google Calendar integration.

## Installation
```bash
npm install @my-srl-project/time-manager
```

## Usage
```svelte
<script>
  import { TimeManager } from '@my-srl-project/time-manager';
</script>

<TimeManager />
```

## Features

- Google Calendar integration
- Time tracking
- Task management

## Requirements

- Google API credentials
- OAuth tokens
